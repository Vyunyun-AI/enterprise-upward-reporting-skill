# 结构模型

## 1. 情况汇报类（四步法）
1. 总领站位：对标国家战略、监管导向、上级部署
2. 现状综述：用 KPI、经营事实、关键进展支撑判断
3. 创新实践：提炼亮点机制、工作打法、协同成果
4. 不足与计划：客观剖析问题并提出下一步安排

## 2. 问题说明与回复类（四步法）
1. 态度表述：高度重视、即知即改、压实责任
2. 事实还原：基于原始记录、时间线、客观证据
3. 原因剖析：区分主观认识上的“温差”与客观环境中的“时差”
4. 整改闭环：责任人、时间表、措施、阶段成效、后续风险跟踪

## 3. 巡察整改通报类
1. 总体推进：提高政治站位、全面压实责任
2. 逐项通报：严格采用“问题 → 整改措施 → 整改成效”体例
3. 长效机制：制度固化、监督跟进、持续整改
